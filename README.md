# Автомобильная панель приборов (Dashboard)

Проект интерактивной панели приборов, разработанный на C++ и Qt/QML с использованием системы сборки **CMake**.

---

## 🛠 Требования к системе

Для успешной сборки и запуска проекта на **Windows** или **Linux** необходимо установить:

1. **Qt SDK** — рекомендуется версия **Qt 6.x** (убедитесь, что при установке выбраны компоненты `Qt Quick` и `Qt5Compat`, если используются старые графические эффекты).
2. **Qt Creator** — или любая другая IDE с поддержкой CMake (например, VS Code или CLion).
3. **Компилятор**:
   * **На Windows**: MSVC (Visual Studio) или MinGW.
   * **На Linux**: GCC или Clang.
4. **Инструменты сборки**: `CMake` (версии 3.16 и выше) и `Ninja` / `Make`.

---

## 🚀 Инструкция по запуску

### Вариант 1. Запуск через Qt Creator (Рекомендуемый)

1. **Скачайте проект:**
   * Склонируйте репозиторий через Git или распакуйте архив с исходным кодом в удобную директорию.
2. **Откройте проект:**
   * Запустите **Qt Creator**.
   * Выберите в меню: **File** -> **Open File or Project...** (Открыть файл или проект).
   * Укажите путь к файлу **`CMakeLists.txt`** в корневой папке проекта.
3. **Настройка комплекта (Configure Project):**
   * Выберите необходимый Kit (например, *Desktop Qt 6.x.x GCC/MSVC*).
   * Нажмите синюю кнопку **Configure Project**.
4. **Запуск:**
   * Нажмите кнопку **Run** (зеленый треугольник в левом нижнем углу) или используйте горячие клавиши:
     * `Cmd + R` (на macOS)
     * `Ctrl + R` (на Windows / Linux)

### Вариант 2. Сборка через терминал (Консоль)

```bash
# 1. Перейдите в папку с проектом
cd path/to/dashboard

# 2. Создайте и перейдите в папку для сборки
mkdir build && cd build

# 3. Сгенерируйте файлы сборки с помощью CMake (укажите путь к вашей Qt, если она не в PATH)
cmake .. -DCMAKE_PREFIX_PATH=/путь/к/вашей/папке/Qt/номер_версии/compiler/

# 4. Скомпилируйте проект
cmake --build .

# 5. Запустите исполняемый файл
# На Linux:
./dashboard
# На Windows:
dashboard.exe
```
